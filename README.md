# ebay-sdk

This is a new generator-backed Python SDK prototype for eBay APIs.

Design goals:

- Pydantic v2 request/response models
- `httpx` sync and async transports
- generated REST resource clients from eBay OpenAPI specs
- generated Pydantic v2 models via `datamodel-code-generator`
- small hand-written runtime core for auth, headers, retries, errors, and uploads
- Pythonic namespace layout, for example `client.buy.browse.get_item(...)`

Raw eBay OpenAPI specs are copied into `specs/ebay`. The generator preprocesses those specs
with explicit eBay compatibility patches, writes normalized specs to
`src/ebay_sdk/generated/specs`, generates models with `datamodel-code-generator`, and keeps the
HTTP resource surface generated by the local script.

Regenerate clients:

```bash
uv run --extra dev scripts/generate_openapi.py
```

Run tests:

```bash
uv run --extra dev ty check
uv run --extra dev ruff check .
uv run --extra dev pytest
```
